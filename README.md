# PRCP-1001: Rice Leaf Disease Detection
Build model which can classify the three major attacking diseases of rice plants like leaf blast, bacterial blight and brown spot

📌 Problem Statement

Task 1

- Prepare a complete data analysis report on the given data

Task 2

- Create a model which can classify the three major attacking diseases of rice plants:
  - Leaf Blast
  - Bacterial Blight
  - Brown Spot

Task 3

- Analyze various techniques like Data Augmentation and create a report on that

---

📂 Dataset Information

- This dataset contains 120 JPG images of disease-infected rice leaves
- The images are grouped into 3 classes based on the type of disease
- Each class contains 40 images

🌱 Classes

- Leaf Smut
- Brown Spot
- Bacterial Leaf Blight
